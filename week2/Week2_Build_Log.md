## 🤖 AI 协作记录（7/15）

### 本周 AI 帮了什么

| 任务 | AI 做了什么 | 占比 |
|------|-----------|------|
| 抓取 Monad Docs 结构 | 浏览器自动导航、提取页面快照 | 100% |
| 品牌叙事撰写 | 基于四点架构输出完整叙事框架和文案 | ~90% |
| 海报 Prompt 设计 | 生成三套 Prompt（英文/底图/中文） | ~95% |
| 排版 & 章节组织 | 自动组织 Markdown 结构 | ~90% |
| Week 2 Build Log 模板 | 预填框架（Prompt/资料/错误/判断/计划） | ~85% |
| 笔记同步 | 将内容增量写入 zz-0816.md | ~95% |

### 人类删改 & 核查了什么

| 动作 | 内容 | 原因 |
|------|------|------|
| ✂️ 删除 | 第一版海报 Prompt 过于抽象，缺少叙事文字 | 用户要求"叙事包含在海报内" |
| ✏️ 修改 | 海报从纯视觉 → 图文一体版（含排版） | 理解到 AI 生图工具对中文文字不稳定 |
| 🔍 核查 | 叙事中的四个组件描述是否与文档一致 | 避免幻觉 |
| 🔍 核查 | Agentic Payments 是否确为文档核心功能 | 确认后在笔记中记录"不是周边，是核心" |
| ✏️ 调整 | 文件存放位置从 week1 → week2 | 时间分类 |
| ✏️ 调整 | 海报排版布局（左边数据、右边叙事） | 信息层次需要人工判断 |

### 本周总结：哪些不能交给 AI

| 不能交给 AI 的事 | 为什么 |
|-----------------|--------|
| **判断信息重要性** | AI 抓了整页文档，但"Agentic Payments 是核心功能"这个判断是我读完后自己做出的 |
| **海报文字排版** | AI 给的中文排版是我手动调整的（哪句话在左、哪句在右），AI 只能给建议，最终信息层次需要人定 |
| **叙事语调** | "当交互足够便宜，机器间的经济就诞生了" 这句是 AI 写的，但「保留它」还是「换成更技术的说法」是我判断的 |
| **文件组织** | 文件放在 week1 还是 week2，这个结构逻辑 AI 猜不准 |
| **Git push 时机** | GitHub 连不上要切换 Windows 端推送，这是实操判断 |
